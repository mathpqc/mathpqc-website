# mathpqc-website

This repository hosts a simple static website for MathPQC documents.

## Structure

- `index.html` - landing page
- `MathPQC- AC26 affiliated event.pdf` - document

## Deploy to GitHub Pages

1. Push this folder to the repository.
2. In repository Settings → Pages, choose the `main` branch and `/ (root)` as the publishing source.
3. Your site will be available at `https://mathpqc.github.io/mathpqc-website/`.
